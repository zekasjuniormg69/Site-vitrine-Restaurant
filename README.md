# G-Kub - Site vitrine de restaurant

Site vitrine créé pour le restaurant **G-Kub** avec HTML, CSS et JavaScript.

## Structure du projet

- `index.html` : page principale du site.
- `css/style.css` : styles pour les sections, le responsive et l'expérience visuelle.
- `js/script.js` : comportements interactifs (menu mobile, animations, navigation).
- `IMG/` : dossier d'images contenant le logo, les photos des plats et les flyers.

## Fonctionnalités

- Design professionnel et responsive.
- Menu de navigation fixe et mobile.
- Sections claires : accueil, présentation, carte, plats, galerie et contact.
- Animation d'apparition progressive au défilement.
- Footer avec informations et moyens de contact.

## Installation

1. Copier le dossier du projet sur votre machine.
2. Ouvrir `index.html` dans un navigateur moderne.

## Notes pour un développeur junior

- Modifier les textes dans `index.html` et ajouter de nouvelles images dans `IMG/`.
- Les styles globaux sont définis dans `css/style.css`.
- Le script `js/script.js` gère l'ouverture du menu mobile et les animations.
- Pour ajouter une nouvelle section, garder la structure `section.section` et ajouter la classe `reveal` pour l'animation.
